# Attendance Token (ERC20)

Attendance Token is a basic ERC20 smart contract developed using Solidity and deployed on the SCAI Mainnet through Remix IDE. The token is designed to reward attendance, participation, and internship activities.

---

## Token Information

| Property | Value |
|----------|-------|
| Token Name | Attendance Token |
| Symbol | ATT |
| Standard | ERC20 |
| Decimals | 18 |

---

## Features

- ERC20 Token Creation
- Token Transfer Functionality
- Balance Tracking
- Smart Contract Deployment on Blockchain

---

## Technologies Used

- Solidity
- Remix IDE
- MetaMask
- SCAI Mainnet

---

## Deployment Process

1. Open Remix IDE  
2. Create `AttendanceToken.sol`  
3. Paste the smart contract code  
4. Compile the contract using Solidity `0.8.x`  
5. Connect MetaMask using Injected Provider  
6. Enter the initial token supply  
7. Deploy the contract on SCAI Mainnet
